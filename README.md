# <h1 style = "font-size:3rem;color:orange;">Sprint_5_project

## <h1 style = "font-size:2rem;color:purple;">Proyecto del sprint 5 del bootcamp tripleten.

### <h1 style = "font-size:1rem;color:green;">Descripción del proyecto

El proyecto consiste en la creación de una aplicación para la lecutra de una base de datos de vehículos usados en Estados Unidos. Está aplicación fue creada desde Visual Studio Code, el cual está enlazado con GitHub. La aplicación se va a poder visualizar por medio de streamlit y render.

La aplicación creada permite al usuario interactuar con histogramas el mismo que contiene la información de la cantidad de vehiculos y el kilometraje. También con grafico de dispersión que contiene la información de los años de fabricación de los vehiculos y el precio de venta.

En el siguiente link puedes acceder para interactuar con el proyecto en la aplicación en render:

https://sprint-5-project-ohus.onrender.com
